## ✨ Descripción
**Luis Macías** es un entrenador de **SmartFit**.  
Esta página web brinda toda la información esencial sobre su trabajo, sus planes y su metodología de entrenamiento, con un enfoque práctico y fácil de usar.

Incluye:
- **Chatbot de ayuda básica** para resolver dudas comunes sobre rutinas.  
- **Sección de psicología de ventas** enfocada en mejorar la motivación y el compromiso del usuario.  
- **Planes de entrenamiento personalizados** para diferentes objetivos.  
- **Perfil profesional completo** del entrenador.

---

## 🚀 Funcionalidades principales
- Información clara de **rutinas de gimnasio**.  
- **Chatbot interactivo** de soporte básico.  
- Psicología de ventas aplicada al fitness.  
- **Planes y asesorías personalizadas** según el objetivo del cliente.  

---

## 🛠 Tecnologías utilizadas
- **Frontend:** HTML5, CSS3, JavaScript  
- **Integraciones:** WhatsApp API (contacto directo)  
- **Chatbot:** JavaScript básico + flujo simple
